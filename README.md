#  Reloj Digital vía Wifi hihihi

Un reloj digital moderno de escritorio con visualización de fecha en tiempo real. Este proyecto fue desarrollado para profundizar en la lógica de interfaces gráficas.

## Tecnologías y Conceptos Aprendidos
- **Python 3** como lenguaje principal.
- **PyQt6** Para montar interfaces de escritorio desde cero.
- **Programación Orientada a Objetos (POO):** Refactorización de la estructura heredando directamente de `QWidget` para optimizar el rendimiento y la limpieza del código.
- **QTimer**  Para controlar el refresco de la pantalla cada segundo (1000ms) de forma asíncrona.
- **Maquetación con Estilos (CSS):** Personalización básica (`QFont`), colores y alineaciones implementadas mediante `.setStyleSheet()`.
